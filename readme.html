The NEURON and Python simulation and data analysis code for the paper:

Buccino AP, Kordovan M, Ness TV, Merkt B, Hafliger PD, Fyhn M, Cauwenberghs G, Rotter S, Einevoll GT (2018) Combining biophysical modeling and deep learning for multielectrode array neuron localization and classification. J Neurophysiol 120:1212-1232 http://dx.doi.org/10.1152/jn.00210.2018

is available at the author's github repository: https://github.com/CINPLA/NeuroCNN

The latest verions of LFPy is also needed. There are installation instructions at the github page above.
